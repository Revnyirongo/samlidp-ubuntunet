# Security Policy

## Supported Versions

| Version | Supported |
| --- | --- |
| 1.0.x | Yes |

## Reporting a Vulnerability

Please report security issues privately to:

- revelation.nyirongo@ubuntunet.net

When reporting, include:

- affected version
- deployment model
- exact steps to reproduce
- impact assessment
- relevant logs or request IDs

Please do not open public GitHub issues for suspected vulnerabilities.

## Response Expectations

- Initial acknowledgement target: 3 business days
- Triage target: 7 business days
- Fix timeline: depends on severity and operational risk

## Disclosure

Coordinated disclosure is preferred. A public advisory should wait until:

- a fix or mitigation is available, or
- operators have had reasonable time to apply it
